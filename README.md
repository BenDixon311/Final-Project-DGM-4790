## Setting up the server for GraphQL pages

While in the project directory, open a terminal and run

 `yarn install`
 
 To start docker, run
 
 `yarn dockerStart`
 
 To deploy the Prisma API run
 
 `prisma deploy`
 
 seed the database by running
 
 `yarn seed`
 
 start the server by running 
 
 `yarn start`
